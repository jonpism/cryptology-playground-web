**Cryptology Playground** is a comprehensive, client-side toolkit designed for learning, testing, and analyzing cryptographic primitives. It runs entirely in your browser, meaning no data is ever sent to a server, ensuring total privacy during experimentation.

**[Live Demo](https://jonpism.github.io/cryptology-playground-web/)**

---

## Local Development

Ensure you have **Node.js** installed.

### 1. Clone the Repository
```bash
git clone [https://github.com/YOUR_USERNAME/cryptology-playground-web.git](https://github.com/YOUR_USERNAME/cryptology-playground-web.git)
cd cryptology-playground-web
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run
```bash
npm run dev
```

## Contributions

Feel free to open an issue or a pull request if you find a bug or want to contribute!

### 1. Fork the project
Click the "Fork" button at the top right of this page to create your own copy of the repository.

### 2. Create your Feature Branch
Create a new branch for your specific tool or bug fix.
```bash
git checkout -b feature/EncodeTool
```

### 3. Commit your changes
Make your changes and commit them with a clear message.
```bash
git commit -m "Add some EncodeTool"
```

### 4. Push to the Branch
Push your new branch to your forked repository.
```bash
git push origin feature/EncodeTool
```

### 5. Open a Pull Request
Go to the original repository on GitHub and open a Pull Request (PR) from your new branch.
